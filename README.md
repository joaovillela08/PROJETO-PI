📌 Descrição
Projeto da disciplina Projeto Integrador com objetivo de criar um sistema de eleição.

🎯 Objetivo
Desenvolver um sistema que com base no numero de eleitores, candidatos e pessoas que irão votar, gere um resultado em um dashboard. Cadastro de candidatos Cadastro de eleitores Votação Apuração dos votos Resultado final.
